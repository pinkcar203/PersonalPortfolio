# Personal portfolio

Next.js portfolio site with static export and automated deploy to GitHub Pages.

## View on GitHub Pages

After a successful deploy, the site is available at 

**https://pinkcar203.github.io/PersonalPortfolio/**

Open that URL in a browser.



## Run locally

```bash
npm install
npm run dev
```

Then open **http://localhost:3000/** — in development, `basePath` is off so the app is served at the site root.

```bash
npm run build
```
